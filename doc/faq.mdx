# Frequently Asked Questions

## Account Management
**Q: How do I create an account?**
A: Sign up using your GitHub account for instant access.

**Q: Can I change my account email?**
A: Yes, update it anytime in your account settings.

## Project Funding
**Q: How are funds distributed?**
A: Funds are released to creators after successful project completion and funder approval.

**Q: What happens if a project doesn't reach its funding goal?**
A: All funds are returned to contributors if a project doesn't meet its goal by the deadline.

## Technical Issues
**Q: How do I report a bug?**
A: Use our [GitHub Issues](https://github.com/openwave/issues) page or contact support.

**Q: Is there an API for developers?**
A: Yes, our public API is available for integration. See our [API Documentation](/doc/api-reference).

## Payments
**Q: What payment methods do you accept?**
A: We accept credit cards, PayPal, and cryptocurrency payments.

**Q: Are there transaction fees?**
A: We charge a 5% platform fee on successful funding transactions.

*This FAQ should be regularly updated based on user feedback and platform changes.*
